# Dental Radiography Classification

This is a repository which implements ablation study over the code available at [Dental X-ray Classification Transfer Learning](https://www.kaggle.com/code/dipuk0506/dental-x-ray-classification-transfer-learning/notebook).

I mainly use this repository to be able to integrate with Google Colab and do my processing there.

The database used for this code is available [here](https://www.kaggle.com/datasets/imtkaggleteam/dental-radiography)
